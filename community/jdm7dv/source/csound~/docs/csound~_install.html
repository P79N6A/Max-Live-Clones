Instructions for earlier versions are contained within their respective archive files.

Installing Csound5

csound~ depends on Csound5.

You can download Csound5 from:

     http://sourceforge.net/projects/csound

Consult this table to see which Csound5 version to install:

MacOSX: WindowsXP:
(intel) 5.10 and later
(ppc ) 5.10 and later 		5.10.1 and later


Mac OS X (intel/ppc)
1 Download and install Csound5.
2 Download and expand the csound~ zip file.
3 Navigate to "binaries/MacOSX/" folder.

For MaxMSP 4.5.x on ppc, copy csound~_ppc_float_MaxMSP45.mxo to "/Library/Application Support/Cycling '74/externals/".

For MaxMSP 4.6.x on ppc, copy csound~_ppc_float.mxo to "/Applications/MaxMSP 4.6/Cycling'74/externals/".

For MaxMSP 4.6.x on intel, copy csound~_intel_xxxx.mxo to "/Applications/MaxMSP 4.6/Cycling'74/externals/".

For MaxMSP 5.x.x on ppc, copy csound~_ppc_xxxx.mxo to "/Applications/Max5/Cycling '74/msp-externals".

For MaxMSP 5.x.x on intel, copy csound~_intel_xxxx.mxo to "/Applications/Max5/Cycling '74/msp-externals".

'xxxx' = 'float' or 'double' (Use the one that matches your Csound 5 version.)

Rename the file csound~.mxo.

4 Navigate to "help_files/" folder.

Copy all .csd files, csound~.help to the help folder.  

For MaxMSP 4.5, that folder is: /Applications/MaxMSP 4.5/max-help/

For MaxMSP 4.6, the help folder is: /Applications/MaxMSP 4.6/max-help/

For MaxMSP 5.0 (and later), the help folder is: /Applications/Max5/Cycling '74/msp-help/
If you have previously installed csound~.mxo, make sure that only one copy of csound~.mxo exists in Max's search path. Otherwise, you may accidently load an older version.

If you have prevously installed Matt Ingalls' csound~, be sure to remove it from the externals folder.

To check your installation, open the patch "csound~.help". In the Max window, you should see the csound~ version. It will look something like this:

     csound~ v1.0.7

If you don't see it, then you have not installed csound~.mxo properly, or Matt Ingalls' csound~ is being loaded instead of csound~.mxo.

Never put csound~.mxo in the same directory as your MaxMSP patch.


WindowsXP/Vista
1 Download and install Csound5.
2 Download and expand the csound~ zip file.
3 Navigate to the "binaries\WinXP\" sub-directory.

a) If your Csound5 version number ends in "f", choose csound~_float.mxe.
If your Csound5 version number ends in "d", choose csound~_double.mxe.

Copy csound~_xxxxxx.mxe to your MaxMSP externals folder.

For MaxMSP 4.5, that folder is: C:\Program Files\Common Files\Cycling '74\externals\

For MaxMSP 4.6, the externals folder is: C:\Program Files\Cycling '74\MaxMSP 4.6\Cycling '74\externals\

For MaxMSP 5.0 (and later), the externals folder is: C:\Program Files\Cycling '74\Max 5.0\Cycling '74\msp-externals\
b) Rename the file csound~.mxe.
4 Copy all .csd files and csound~.help to the help folder.

For MaxMSP 4.5, that folder is: C:\Program Files\Cycling '74\MaxMSP 4.5\max-help\

For MaxMSP 4.6, the help folder is: C:\Program Files\Cycling '74\MaxMSP 4.6\max-help\

For MaxMSP 5.0 (and later), the help folder is: C:\Program Files\Cycling '74\Max 5.0\Cycling '74\msp-help\

If you have previously installed csound~.mxe, make sure that only one copy of csound~.mxe exists in Max's search path. Otherwise, you may accidently load an older version.

To check your installation, open the patch csound~.help. In the Max window, you should see the csound~ version. It should look something like this:

     csound~ v1.0.7

If you don't see it, then you have not installed csound~.mxe properly.

Never put csound~.mxe in the same directory as your MaxMSP patch.
